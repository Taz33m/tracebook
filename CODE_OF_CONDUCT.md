# Code Of Conduct

This project should be a place where people can do careful technical work without needless friction.

## Expected Behavior

- Be respectful and specific.
- Assume good intent, but accept correction when impact differs from intent.
- Critique code and ideas, not people.
- Keep disagreements focused on evidence, tradeoffs, and project goals.
- Make space for new contributors to ask basic questions.

## Unacceptable Behavior

- Harassment, threats, insults, or discriminatory language.
- Publishing private information without consent.
- Repeatedly derailing issues or pull requests.
- Bad-faith demands for unpaid support.

## Enforcement

Maintainers may edit, hide, or remove comments; close issues; reject pull requests; or block users when behavior harms the project. If you need to report conduct privately, contact the maintainer through the GitHub profile for `Taz33m`.
